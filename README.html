Code Society 25.2 Cohort

Purpose

Main project repo for the Code Differently 25.2 cohort. Read the syllabus for more information about the class. Use the lesson folders to find and submit homework assignments.

Key locations

Here are some key resources you can use to more easily navigate this repo:

  1. syllabus/: The class syllabus containing everything you need to know about topics and policies for the class.
  2. slides/: The HTML-based lecture slides used for class. These are deployed via GitHub pages at code-differently.github.io/code-society-25-2/slides
  3. lesson_##/: Each lesson will have it's own folder and README at the root of the repo (e.g. lesson_00/). Make sure that you are in the right lesson folder when running commands and scripts for homework.
  4. project_oop/:: Once published, you'll find the midterm project requirements here.
  5. capstone/: Once published, you'll find capstone project requirements here.
  6. lib/: A repository of shared code and sample projects demonstrating techniques learned throughout the class.
